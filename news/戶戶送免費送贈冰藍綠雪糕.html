戶戶送免費送贈冰藍綠雪糕

  • 2 August 2018
  • Deliveroo
  • 閱讀需時: 0 min
teal-ice-cream
分享此文章
  • 為炎夏消暑降溫並玩轉Instagram

香港正值炎炎夏日，驕陽似火，悶熱潮濕。為了讓大家消暑降溫，本港美食外賣速遞公司戶戶送特別夥拍城中人氣高企的意式餐廳酒吧208 Duecento Otto，送出外形吸引、味道出眾的冰藍綠雪糕，定必令一眾雪糕迷及 Instagram愛好者心花怒放，暑氣全消！

戶戶送將於8月8日至10日正午12時至下午2時30分在上環、灣仔及中環的熱鬧街頭，向途人免費送上美麗怡神的冰藍綠雪糕，讓上班族為午餐時段畫上甜蜜句號。粉彩色系向來是夏日潮流之選，而這款名為Roo Velvet的忌廉芝士味雪糕不僅顏色嬌美迷人，味道亦非同凡響，定能讓你玩轉 Instagram！Roo Velvet每人限送一杯，送完即止。

8月8日（星期三） //  正午12時至下午2時30分    //  上環蘇杭街

8月9日（星期四）//   正午12時至下午2時30分    //   灣仔莊士敦道  (港鐵灣仔站A3出口)

8月10日（星期五）//  正午12時至下午2時30分  //   中環閣麟街7-11門外

208 Duocento Otto 大廚Mick Bolam表示：「在香港，炎夏的高溫令人難以忍受。在悶熱天氣下，很多人亦會食欲不振，以冰涼透心的雪糕作為午餐絕對可以接受。這款冰藍綠雪糕Roo Velvet的表面鋪上一層忌廉芝士，散發的甜蜜滋味足以媲美深受歡迎的紅絲絨蛋糕 ，只是顏色截然不同，換上了戶戶送商標所用的冰藍綠色。」

即使錯過了今次免費享用雪糕的機會亦無須失望，因為這款清新怡神的冰藍綠雪糕將在戶戶送平台上以供點選。現在就以戶戶送的冰凍雪糕為炎夏消暑降溫！
